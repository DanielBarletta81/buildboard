## 🎯 Interactive Behaviors

### Checkbox Interactions
```
State:   ☐ Unchecked  →  [Click]  →  ☑ Checked
Visual:  Gray border  →           →  Green border
         Normal text  →           →  Strikethrough
         No icon      →           →  Checkmark icon
```

### Shopping List Interactions
```
State:   Not in List     →  [Click]  →  In List
Button:  [+ Add to List] →           →  [✓ In Shopping List]
Color:   Gray/Subtle     →           →  Gold/Prominent
Badge:   🛒 Shopping (0) →           →  🛒 Shopping (1)
```

### Cost Tracking Interactions
```
State:   Estimated      →  [Click]  →  Editing       →  [Enter]  →  Saved
Display: ~$19.99 [track] →           → [input: 20.50] →          → $19.99 (Actual: $20.50)
Budget:  $425 / $3,500  →           →  $425 / $3,500  →          → $445.50 / $3,500
```

### Filter Interactions
```
Filter:  [All]     →  Shows all cards (completed + pending)
         [Pending] →  Hides completed cards, shows only incomplete
         [Completed] → Hides pending cards, shows only finished
```

## 🎬 User Flows

### Flow 1: Material Purchase
```
1. Navigate to Materials board
2. See list of all materials needed
3. Click "Pending" filter to see what's not bought
4. Click "+ Add to List" for each needed item
5. Click 🛒 Shopping List badge
6. Review grouped items by supplier
7. Click "Copy List" to clipboard
8. Shop online or in-store
9. As you purchase, mark items complete (☑)
10. Track actual costs
11. Watch budget and progress update
```

### Flow 2: Building Progress
```
1. Navigate to Steps board
2. View first incomplete step
3. Check materials/tools needed
4. Complete the step
5. Click step number to mark complete (✓)
6. See progress dashboard update
7. Move to next step
8. Repeat until project complete
9. See 🎉 celebration!
```

### Flow 3: Budget Management
```
1. View progress dashboard
2. See estimated budget
3. As you buy materials, track actual costs
4. Watch running total update
5. Get warning if over budget (red text)
6. Adjust purchases or budget accordingly
```
